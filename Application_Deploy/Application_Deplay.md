# Application 배포

## Application 배포란 무엇인가요?

- 애플리케이션 배포 환경을 구성하고, 구현이 된 애플리케이션의 소스 검증 및 빌드를 수행하여, 운영체제 환경에 배포하는것을 말한다.
- 즉 코딩해서 서버에 올리기 까지의 과정 !!

<img src="../Images/Deploy.png" alt="image12" width="600">

### 개발 서버 (Development Server)

- 개발 테스트가 이루어지는 서버이며, 테스트용 DB를 가지고 있다.

### Staging Server

- 실제 운영 환경과 거의 동일한 환경을 세팅 해 놓고 운영환경으로 이전하기 전에 다양한 기능을 테스트 하는 환경이다.

### Product Server

- 사용자가 직접 사용하는 서비스 운영 환경이며, 실제 서비스가 돌아가는 서버이다.
